## 两数之和
给定一个整数数组 `nums` 和一个目标值 `target`, 请你在数组中找出和为目标值的那 **两个**整数, 并返回他们的数组下标.
你可以假设每种输入只会
```js
function sums(arr, resNum) {
	var mMap = {}, res = [];
	for(var i = 0; i < arr.length; i++) {
		mMap[arr[i]] = i;
	}
	for(var i = 0; i < arr.length; i++) {
		var t = resNum - arr[i];
		if(mMap[t] && mMap[t] != i) {
			res = [arr[i], t];
			break;
		}
	}
	return res;

}
```
